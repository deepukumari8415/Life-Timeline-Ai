## 💡 About This Project
Life Timeline AI helps students visualize their educational journey based on their current course and date of birth.

The application provides:
- 🎓 Career roadmap
- 📅 Graduation year prediction
- 💼 Career suggestions
- 📚 Skills to learn
- 🔥 Motivation based on career path
